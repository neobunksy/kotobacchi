# 子供向け学習アプリ 要件定義書 v2.0

## アプリ名（仮）
ことばっち

---

## コンセプト
AIおともだちと一緒に遊びながら
- ひらがな
- カタカナ
- アルファベット
- 音
- 会話

を学ぶ知育アプリ。

「教育アプリ」ではなく「AIおともだちと遊ぶアプリ」として設計する。

## 初回起動

ユーザー名登録

AIおともだち「おなまえをおしえてね」→ 入力 → 保存

以後「りゅくん」「りゅちゃん」として呼びかける

---

## AIおともだち

### 概要
アプリ全体のナビゲーター。全画面に常駐。

### 役割
- 会話相手
- クイズ出題
- 応援・褒める
- 着せ替え対象

### 表示位置
右下固定。タップ時→会話画面を表示。

### AI会話ルール
- 回答文字：ひらがな・カタカナのみ
- 禁止：漢字・難しい単語・長文
- 最大50文字

会話例：
- ユーザー「いぬはなにをたべるの？」
- AI「いぬは ごはんを たべるよ」

## ホーム画面メニュー

3. おともだちとおはなし
4. きせかえ

---

## もじであそぶ

### 画面構成
- 上部：入力エリア
- 中央：文字表
- 下部：クイズボタン

### 表示切替
- ひらがな
- カタカナ
- ABC

### 文字ボタン押下時
1. 文字表示
2. 音声再生

### 音声設定
初期状態：ON（すべての文字で音声再生）

### 対応文字

**ひらがな五十音全対応**

濁点：が ぎ ぐ げ ご / ざ じ ず ぜ ぞ / だ ぢ づ で ど / ば び ぶ べ ぼ

半濁点：ぱ ぴ ぷ ぺ ぽ

拗音：きゃ きゅ きょ / しゃ しゅ しょ / ちゃ ちゅ ちょ / にゃ にゅ にょ / ひゃ ひゅ ひょ / みゃ みゅ みょ / りゃ りゅ りょ / ぎゃ ぎゅ ぎょ / じゃ じゅ じょ / びゃ びゅ びょ / ぴゃ ぴゅ ぴょ

小文字：ぁ ぃ ぅ ぇ ぉ / ゃ ゅ ょ / っ

長音：ー

**カタカナ**：ひらがなと同等対応

**アルファベット**：A〜Z（押下時→「エー」など読み上げ再生）

---

## クイズモード

別画面なし。もじであそぶ画面内で実行。

### 流れ
1. クイズボタン押下
2. AI「"さ" をおしてね！」
3. 正解→「せいかい！」
4. 不正解→「もういちど！」

### 難易度
- レベル1：あいうえお
- レベル2：全五十音
- レベル3：濁点
- レベル4：小文字
- レベル5：アルファベット

---

## おとであそぶ

### モード切替
- ピアノ：ドレミファソラシ
- ドラム：キック・スネア・ハイハット・シンバル
- 動物：犬・猫・牛・馬・豚・羊・ライオン・象

押す→音再生
